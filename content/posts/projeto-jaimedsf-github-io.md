Repositório: [github.com/Jaimedsf/jaimedsf.github.io](https://github.com/Jaimedsf/jaimedsf.github.io)

Este é o repositório do **blog que você está lendo agora**. Construído com [Hugo](https://gohugo.io/) + tema [Hextra](https://imfing.github.io/hextra/), publicado automaticamente via GitHub Actions no GitHub Pages.

Inspirado no [post do Akita Onrails](https://akitaonrails.com/2025/09/10/meu-novo-blog-como-eu-fiz/), o fluxo é: escrever markdown em `content/posts/`, dar push, e o site é gerado e publicado sozinho.
